## Main User Journey

1. Open the frontend and log in.
2. Upload a video with title, description, and category.
3. Watch realtime processing progress in the dashboard.
4. Use filters to refine the video library by metadata.
5. Select a video to review metadata and stream playback.

## Admin Flow

Admins can:

- view tenant users
- create additional users
- manage tenant categories

## Editor Flow

Editors can:

- upload and review videos
- create categories for the tenant

## Viewer Flow

Viewers can:

- log in
- review only their accessible videos
- stream and inspect processed video entries
